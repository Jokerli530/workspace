## 外卖运营核心知识体系

### 一、流量运营

**新店冷启动（前30天）**
- 开业前3天：积累基础评价（5-10条带图好评）
- 第1周：满减活动拉新（建议客单价5-8元利润空间）
- 第2-4周：梯度优惠留存（满25减5 → 满35减5）
- 评分维护：≥4.5分是安全线，<4.0会降权

**排名权重（美团）**
1. 销量（最近30天）— 权重最高
2. 评分 + 好评率
3. 客单价匹配度
4. 配送距离
5. 活动丰富度

**流量入口**
- 频道页（快餐/夜宵/甜点等）
- 搜索关键词（店铺名+品类名）
- 推荐页（平台算法）
- 会员频道（高价值用户）

### 二、活动设计

**满减（最核心）**
- 门槛 = 客单价 × 1.2~1.5
- 力度：参考同商圈竞品
- 套路：设置虚高原价 + 大折扣，制造"超值感"

**新客立减**
- 首单减5-10元，获取新客

**满返券**
- 满50返5元券，刺激复购

**品类券**
- 特定品类（如奶茶、火锅）专属券

### 三、菜单运营

**菜单结构**
1. 引流款（1-2个：低价高频）
2. 主力款（3-5个：利润担当）
3. 凑单款（饮品、小食）
4. 套餐款（提升客单价）

**菜品优化**
- 命名：品类词 + 特色词 + 口味词
  - ✅ "招牌麻辣香锅（微辣）含米饭"
  - ❌ "麻辣香锅"
- 图片：明档实拍 > 效果图
- 描述：交代食材、份量、口味（辣度/甜度）

### 四、评分管理

**好评引导**
- 小票随餐好评返券（1-2元）
- 差评48小时内响应和解

**差评分类处理**
| 类型 | 应对 |
|---|---|
| 配送慢 | 联系骑手/平台，补偿顾客 |
| 口感问题 | 改进包装 or 调整配方 |
| 份量不足 | 增加份量 or 调整描述 |
| 食材问题 | 下架/更换供应商 |

### 五、数据指标

**核心指标**
| 指标 | 合格线 | 优秀 |
|---|---|---|
| 店铺评分 | ≥4.5 | ≥4.8 |
| 单量/天 | 因品类而异 | 持续增长 |
| 客单价 | ≥品类均值 | 高于均值20% |
| 复购率 | ≥15% | ≥25% |
| 曝光转化率 | ≥8% | ≥12% |

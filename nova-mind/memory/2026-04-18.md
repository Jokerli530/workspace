## HiClaw — 企业级OpenClaw多Agent协作系统

**agentscope-ai/HiClaw — 4,165 stars**
"An open-source Collaborative Multi-Agent OS for transparent, human-in-the-loop task coordination via Matrix rooms."

### 关键特性
- **Manager-Workers架构**: Manager统筹多个Workers，人类全程可见可干预
- **明确支持OpenClaw**: 每个Agent可以是OpenClaw/Copaw/NanoClaw/ZeroClaw
- **Matrix协议IM**: Element IM + Tuwunel IM Server，去除钉钉/飞书集成开销
- **Higress AI Gateway**: 集中流量管理，凭证不暴露给Workers（安全!）
- **MinIO共享文件系统**: Agent间信息交换，显著降低token消耗
- **Skills生态**: Workers按需从skills.sh (80,000+ community skills)拉取技能
- **一行安装**: curl | bash 完成所有部署

### 安全模型
- Worker Agents只用consumer tokens
- 真实凭证(API keys, GitHub PATs)留在Gateway里
- Workers看不到凭证，攻击者也拿不到

### 对Nova的启示
- Manager-Workers = OpenMOSS的多Agent协作的成熟形态
- HiClaw的enterprise security model值得借鉴
- skills.sh (80k skills!)是ClawHub的直接竞争者

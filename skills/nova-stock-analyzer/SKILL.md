# Nova Stock Analyzer — A股十维深度分析

## 十维评分体系

| # | 维度 | 满分 | 核心评估点 |
|---|------|------|-----------|
| 1 | 技术面 | 10 | MA排列、布林带、量价、支撑阻力位 |
| 2 | 资金面 | 10 | 主力净流、融资融券、北向、龙虎榜 |
| 3 | 筹码结构 | 10 | 大股东、解禁、质押、股东人数、机构持仓 |
| 4 | 基本面 | 10 | 营收/净利增速、ROE、毛利率、现金流 |
| 5 | 赛道 | 10 | 产业链位置、政策、市场空间、国产替代 |
| 6 | 估值 | 10 | PE/PB/PS、同行对比、机构目标、安全边际 |
| 7 | 风险 | 10 | 实控人、质押、解禁、监管、负面事件 |
| 8 | 事件日历 | 10 | 近期催化、业绩预告、政策节点 |
| 9 | 同业对比 | 10 | 与竞争对手多维度横向比较 |
| 10 | Qlib信号 | 10 | 量化模型得分+排名+IC置信度 |

**总分: 100分**

**判定标准：**
- 60+ → 🔥 建仓候选
- 50-59 → 🟢 重点关注
- 40-49 → 🟡 观察
- <40 → 🔴 回避

## 分析流程

### Phase 1: 快速并行 (1分钟)
```
实时行情: curl hq.sinajs.cn → 价格/量/涨跌
Qlib信号: 拉预测得分+排名(如有)
基本面快照: web_search(同花顺/东方财富)
近期重大事件: web_search
```

### Phase 2: 十维深度 (按顺序)
```
1. 技术面: Qlib拉1年日线 → MA/布林/支撑阻力/量比
2. 资金面: web_search → 主力流/融资/北向
3. 筹码: 同花顺F10 → 大股东/质押/解禁/人数
4. 基本面: 近3年+Q1财务 → 增速/ROE/毛利率/现金流
5. 赛道: web_search → 政策/规模/产业链位置
6. 估值: PE/PB/同行/机构目标/安全边际
7. 风险: 内幕交易/质押比例/退市风险/概念错配
8. 事件: 未来6个月催化剂日历
9. 同业: 横向对比表(市值/PE/赛道/护城河)
10. Qlib: 信号得分+排名+IC置信度
```

### Phase 3: 输出模板
```
## 🔬 XX股 XXXX — 深度剖析

### 一、技术面 + 资金面 + 筹码
### 二、基本面 + 赛道
### 三、估值 + 同业对比
### 四、风险 + 事件日历
### 五、Qlib信号
### 六、十维评分卡
### 七、入场策略(时机+目标+止损)
```

## 批量模式(10+只)
```
轻量版: 只做Phase 1 + Qlib + 快速估分
输出排序表 + 分层(🔥/🟢/🟡/🔴)
然后用户选重点再走完整版
```

## ⚠️ 关键风控
```
必查项:
1. ST/退市: 新浪API实时名称检查(不是Qlib数据)
2. 内幕交易/立案 → 一票否决🔴
3. 概念错配 → 查互动平台/公司声明
4. PE>200 → 标注估值泡沫
5. 涨停>3板 → 警惕游资接力
6. 大股东质押>60% → 强风险标记
```

## 自我进化
```
每次完整分析后记录:
- 哪些维度被证明最关键 → 调整权重
- 新的数据源/查询方式 → 补充数据源表
- 误判案例 → 分析原因，更新风控规则
```
